

## Hosting

1. Push the folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set the source to the main branch and the root folder.
4. Save and wait for the Pages URL to appear.

## Files

- `index.html` is the main entry page.
- `final.html` is the closing page.
- `styles.css` contains the theme and layout.
- `app.js` contains slideshow and quiz logic.
